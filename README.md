## Modelos externos
En esta actividad tuvimos la oportunidad de experimentar el uso de modelos externos para cumplir los requerimientos de una aplicación de traducción y resumición de texto. A nivel técnico, se explora el uso de la libreria OpenAI como interfaz de comunicación con los modelos (ya sea local o en la nube). Por otro lado, mediante la biblioteca Gradio se desarrolló una interfaz para garantizar la usabilidad. 

### Limitaciones
No implementamos modelos locales debido a que nuestras máquinas no cumplen con los requerimientos.
